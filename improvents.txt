1. Incorrect ARIA Roles in Navigation

Problem:
role="menubar" and role="menuitem" are meant for application-style menus, not website navigation. This can confuse screen readers.

Current Code:

<div class="nav-links" role="menubar">
  <a role="menuitem">Home</a>
</div>


Fix:
Use semantic HTML and remove unnecessary ARIA.

<nav aria-label="Main navigation">
  <ul class="nav-links">
    <li><a href="index.html">Home</a></li>
  </ul>
</nav>

2. Inline JavaScript (onclick) in HTML

Problem:
Inline JS mixes structure with behavior and is harder to maintain and scale.

Current Code:

<button onclick="toggleCart()">Cart</button>


Fix:
Move event handling to JavaScript.

<button class="cart-btn">Cart</button>

document.querySelector('.cart-btn')
  .addEventListener('click', toggleCart);

3. Missing Focus Management for Cart Sidebar

Problem:
When the cart opens, keyboard users are not moved into the dialog, breaking accessibility.

Fix:
Move focus to the cart and restore it when closed.

cartSidebar.setAttribute("aria-hidden", "false");
cartSidebar.querySelector("button").focus();


On close:

cartSidebar.setAttribute("aria-hidden", "true");
cartButton.focus();

4. Overly Strict Content Security Policy (CSP)

Problem:
script-src 'self' blocks analytics, payments, and future integrations.

Current Code:

script-src 'self';


Fix:
Allow trusted third-party services.

script-src 'self' https://www.googletagmanager.com;

5. No Lazy Loading for Product Images

Problem:
All images will load immediately, slowing page performance on mobile.

Fix:
Enable native lazy loading.

<img src="soap.jpg" loading="lazy" alt="Lavender Soap">

9. Missing aria-expanded State Updates

Problem:
Hamburger and cart buttons do not update aria-expanded, misleading screen readers.

Fix:
Update state dynamically in JS.

button.setAttribute("aria-expanded", "true");


Set to "false" when closed.

10. No Image alt Strategy Defined

Problem:
Future images may lack meaningful alt text, hurting accessibility and SEO.

Fix:
Use descriptive alt text for products.
